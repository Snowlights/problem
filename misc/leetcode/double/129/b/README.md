### 题目

<p>给你一个二维 boolean 矩阵&nbsp;<code>grid</code>&nbsp;。</p>

<p>请你返回使用 <code>grid</code>&nbsp;中的 3 个元素可以构建的 <strong>直角三角形</strong> 数目，且满足 3 个元素值 <strong>都</strong>&nbsp;为 1 。</p>

<p><b>注意：</b></p>

<ul>
	<li>如果 <code>grid</code>&nbsp;中 3 个元素满足：一个元素与另一个元素在 <strong>同一行</strong>，同时与第三个元素在 <strong>同一列</strong>&nbsp;，那么这 3 个元素称为一个 <strong>直角三角形</strong>&nbsp;。这 3 个元素互相之间不需要相邻。</li>
</ul>

<p>&nbsp;</p>

### 思路

统计每行和每列1的个数，之后计算当元素为1时，符合条件的三角形数量.

``` 
func numberOfRightTriangles(grid [][]int) int64 {

	m, n := len(grid), len(grid[0])
	r, c := make([]int, m), make([]int, n)
	for i, v := range grid {
		for j, vv := range v {
			if vv == 0 {
				continue
			}
			r[i]++
			c[j]++
		}
	}

	ans := 0
	for i, v := range grid {
		for j, vv := range v {
			if vv == 0 {
				continue
			}
			ans += (r[i] - 1) * (c[j] - 1)
		}
	}
	return int64(ans)
}
```

### 复杂度分析

- 时间复杂度：$\mathcal{O}(nm)$，其中 $m$ 为 $\textit{grid}$ 的行数，其中 $n$ 为 $\textit{grid}$ 的列数。
- 空间复杂度：$\mathcal{O}(m+n)$。忽略排序的栈开销。
